2019.11.7

整理大项

分步查询，但感觉并没有学到实际的东西，还是皮毛

找到公众号，准们针对面试进行总结，之后在扩展

类似应试教育，，，从面试题开始，扩散，问的不外乎还是那些东西

  1. 我的优势，在于现在项目中用的是微服务，分布式的开发部署方式，把这块好好搞清楚。包括权限管理，网关，分布式session共享，数据库设计，项目前后端整体开发流程，线上项目排错；
  2. 需要补的就是面试常见的这些东西，优势方面还需要加强。

2019.11.9

今天上午九点半起来，忙完就快十一点了，然后学习到近三点，吃饭吃撑了，然后看视频到六点多，现在在写写日记看看知乎，准备继续学习，身体有点疲惫，同时想到，我觉得我不应该这么菜，这么点学习任务，一直又拖了一周才完成大部分，我不应该这么菜吧。

到八点出去转转，应该把自己的生活安排满一点，这样时间也就能够更加高效节约。

又兜兜转转看看卖ipad的，看到八点十分了。

我发现学习个三四个小时之后，还是要运动一下，不然接下来效率很低，还有在家自学的时候，还是要把番茄工作法用起来的，不然时间过得很快。

分辨率高屏幕真的很清

吃的太多真的很当务事

运动有利于自身的专注

还有早点睡

要吃的清单，有利于专注

品栋数码的要比尾巴贵个至少两百，但是买着很放心（ipad pro9.7）

晚上尽量看半个小时的书，早上运动一下，阶段性的运行是提高

晚上睡前打坐20分钟，早上站桩30分钟，身体好

2019.11.10

把分辨率调到100%，13村的屏幕也很清晰，所以屏幕不清也可能是分辨率的问题。

今天 整理之余要学学五笔打字，放了太久，又得重新学了。

  • spring aop实现
  • hashMap相关，list的排序取值
  • 二十个经典的问题整理
  • 多线程面试题

九点半起来，洗刷收拾，然后去吃包子。

整理了一片唐家三少15年的回复，算是自己经常想起来的东西，找着很麻烦，不如收藏了 😋 😂

刚整理了hashMap的hashCode和equals的用法，已经三四个小时了，已经学不进去了。出去转转，买点鱼煎煎。

问题整理：

一、介绍自己对 spring 的理解？你怎么理解 Ioc，AOP？

二、异常？项目中如何处理异常？运行时异常和受检查异常的区别？为什么要处理异常？

三、你对 dubbo、zookeeper、restful 的理解？以及使用这些的好处？

四、线程？场景举例，我现在记得的是这样的一个例子。有一个含有十个元素的数组，现在有两个线程？如何同时输出前五个元素和后五个元素？如何先输出前五个，再输出后五个？

五、hashMap 的实现原理？concurrentHashMap 的原理？

六、数据库优化怎么做？

七、除了 ssh 这些你还会什么？

关于 http

悲观锁和乐观锁

今天看了这么多，发现对于一些hashCode的实现原理、算法，spring aop实现原理，还是看不懂，一些比较深的东西，看完整理了但效果并不大，还是需要从比较基础的看起，对于一些比较深入的方面，比如spring，看书还是最快的途径，对于多线程的东西，要从是什么，怎么实现看起，而不是一开始就去看决绝哪些案例，问题等等。

还有下周，看书的同时，要开始把自己的项目，项目中用到的技术，项目中遇到的问题，整个整理出来，要对自己做过的项目有足够清晰的认识，这个我觉得还是可以的，问题不大，因为项目中的问题虽然之前疏于整理，但都是思考过得，也要看一看之前的记录。

面试题太多了，看不完的，要有自己的进度，一步一步来。主要是要能有效的输入，想办法把效率提升上来。
